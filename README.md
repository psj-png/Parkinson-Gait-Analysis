#  AI 기반 파킨슨병 조기 진단 및 보행 분석 시스템



> **MediaPipe와 머신러닝을 활용한 비침습적 보행 패턴 분석 프로젝트**







이 프로젝트는 고가의 장비 없이 RGB 영상만으로 파킨슨병의 초기 징후를 포착하는 것을 목표로 합니다. 데이터 노이즈 정제부터 머신러닝 모델 튜닝까지 조기 진단의 신뢰성을 높이기 위한 6단계 파이프라인을 구축했습니다.







---







### 주요 개발 단계 







1. **01_preprocess.py**: MediaPipe를 통해 영상 내 관절 포인트가 정상적으로 추출되는지 실시간으로 검수합니다.



2. **02_extractor.py**: 추출된 좌표의 미세한 떨림을 잡기 위해 이동평균필터와 Butterworth Low-pass Filter를 적용했습니다.



3. **03_labeling.py**: 임상적 유의미성이 높은 **무릎 각도**와 **상체 기울기**를 직접 계산하고, 이를 기반으로 데이터 라벨링을 자동화했습니다.



4. **04_visualizer.py**: 정제된 데이터를 그래프로 시각화하여 정상군과 파킨슨군 사이의 보행 변동성을 검증합니다.



5. **05_trainer.py**: **Random Forest** 모델을 학습시킵니다. 데이터 불균형 문제를 해결하고 조기 진단 민감도를 높이기 위해 **가중치(Class Weight)를 15배** 설정했습니다.



6. **06_diagnosis.py**: 학습된 모델을 로드하여 새로운 영상의 파킨슨 확률(%)을 산출하고 판정 결과를 출력합니다.
